### To run the backend
- To create a virtual python environment
'python -m venv .venv'
- To activate the virtual python environment
'.venv\Scripts\Activate.ps1'
- To Install the required dependencies
'pip install -r requirements.txt'
- To start backend
'uvicorn main:app --reload'

### To run the frontend
- To install the required dependencies
'npm install'
- To run frontend
'npm run dev'
